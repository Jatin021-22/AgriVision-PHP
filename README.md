## 🌱 Overview

**Agri Vision: Growing a Better Tomorrow** is a practical agriculture-focused web application built to support farmers and agri-enthusiasts through a single digital platform.  

The system combines **product purchasing, agricultural news, blogging, and live weather forecasting** to help users make smarter and more timely farming decisions.

---

## ✨ Core Highlights

🔹 Clean and farmer-friendly interface  
🔹 Real-time weather insights for better crop planning  
🔹 Integrated agriculture news updates  
🔹 Community blogging and knowledge sharing  
🔹 Simple product browsing and purchasing flow  
🔹 Lightweight and fast Core PHP architecture
